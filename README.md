# HabitOff – Break Bad Habits, Build Better Ones

HabitOff is a simple, lightweight Android application designed to help users track and reduce harmful habits. The app focuses on clean UI, easy interaction, and daily progress tracking — perfect for users who want to replace bad habits with healthier alternatives.

# Features
1. Simple Habit Tracking

Add any habit you want to quit or control

Track streaks and daily progress

Visual indicators to motivate consistency

2. Analytics & Progress View

Daily, weekly, and monthly insights

Graph-based representation of improvement

3. Smart Reminders

Set reminders to avoid relapse moments

Customizable notification timings

4. Minimal & Clean UI

Smooth animations

Intuitive and distraction-free design

5. Offline Support

Works completely offline
